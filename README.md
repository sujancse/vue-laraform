# Vue Laraform
Vue laraform is for handling laravel form submission and error handling.

![license (scoped)](https://img.shields.io/github/license/sujancse/vue-laraform.svg)

# Installation
```
npm i vue-laraform
```

# Usage
```
const vueLaraform = require("vue-laraform");
```
